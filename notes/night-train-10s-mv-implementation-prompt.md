# 夜车 10s 像素 MV 实现提示词

请使用 `pixel-mv-10s` 的思路，为歌曲 **《夜车》— 曾轶可** 生成一个 **10 秒像素 MV 原型**。

目标不是普通 loop，不是 bots 页面背景，而是一段 **10 秒、带起承转合的像素短片**。

## 核心要求

- 时长：**10 秒**
- 风格：像素风、深夜、中文歌情绪表达优先
- 重点：**夜车车窗 + 月光 + 树影掠过 + 忧伤的翘尾巴幽灵 + 酒瓶 + 钥匙**
- 不要做成高能炫技 MV，但也不要过度克制；这次需要让**物件、人物主体、灯光和环境的变化幅度明显增大**
- 仍然保持固定镜头，但画面中的节奏、摆动、掠过、闪现感要更强

## 核心一句话

一个忧伤的翘尾巴幽灵坐在夜车窗边，尾巴轻轻翘起，手边有酒瓶与钥匙；窗外月光、树影和远处灯光不断掠过，像记忆在车窗外一层层后退。

## 主角设定

主角不是人，而是一个：
- **忧伤的翘尾巴幽灵**
- 身体半透明
- 尾巴向上轻轻弯起，是轮廓识别重点
- 有一点被月光照到就会发亮的质感
- 情绪不是惊吓或诡异，而是：**失神、柔软、没说出口、被夜色拖着走**

## dominant image

- 深夜列车窗边
- 忧伤的翘尾巴幽灵坐在窗边或靠着座位
- 一只酒瓶放在侧边
- 一把钥匙在手边或腿边晃动一下
- 窗外有月光、掠过的树影、偶尔经过的灯光
- 整个世界在窗外向后退，而幽灵被困在这 10 秒里

## motion truth

- primary motion: **pass-by / sway / flicker**
- motion source: **environment-driven + emotional resonance**
- viewing mode: **watched**
- material: **window light / moonlight / glass reflection / tree shadows / bottle glint / key swing / ghost tail**

## 10 秒结构

### 0–2s — Reveal
- 先看见车厢暗面、列车窗和窗边一小块座位
- 窗外有冷月光慢慢铺进来
- 忧伤幽灵轮廓逐渐显现，尾巴先比身体更清楚
- 酒瓶的边缘反一点点光

### 2–5s — Window world intensifies
- 列车开始更明显地晃动
- 窗外树影一片片横向掠过
- 月光与远处灯光交替扫过车窗
- 钥匙出现一次轻微晃动或闪光
- 幽灵尾巴、肩部、头部都出现更明显的轻晃，而不是几乎静止

### 5–8s — Emotional peak
- 一次最强的月光/灯光掠过车窗
- 车窗、树影、月光、酒瓶反光同时叠在一起
- 幽灵像差一点转头、差一点伸手、差一点想说什么
- 尾巴在这一刻明显抬高或摆一下
- 酒瓶或钥匙在这段中至少有一个被强光击中，形成高点细节

### 8–10s — Residual close
- 树影和灯光重新变稀
- 月光还留一点在窗边
- 幽灵重新沉回安静，但尾巴还有余晃
- 酒瓶和钥匙只剩暗暗的一点反光
- 整体回到夜车持续前进的状态，形成可循环余韵

## scene layers

至少包含：
- dark carriage shell
- train window frame
- seat edge / window-side seat
- sad tail-up ghost presence
- passing light bands
- moonlight wash
- tree shadows outside window
- glass reflection
- bottle prop
- key prop

## 视觉和节奏约束

- 镜头固定，不切镜头
- 角色不需要大幅位移，但身体、尾巴、轮廓亮暗、姿态变化都要比上一版明显
- 列车晃动、树影掠过、月光扫过的幅度要更大，让 10 秒更有“正在发生”的感觉
- 酒瓶和钥匙不是纯装饰，要至少各自承担一次视觉提示功能
- 月光是重要角色，不只是环境补光
- 树影不能像普通横条，要像真实窗外树木一簇簇掠过

## 成功标准

这 10 秒看完后，应该让人感觉：
- 这是一个有物件记忆的夜车片段，而不是空车窗
- 幽灵真的有情绪和存在感，尤其是尾巴轮廓足够鲜明
- 月光、树影、钥匙、酒瓶都在参与表达，而不是背景摆设
- 画面变化幅度明显更大，但仍然维持夜里的忧伤和克制
